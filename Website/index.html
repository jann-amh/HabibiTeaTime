Commands

~Chatterino This cmd gives you a link to the current Chatterino version.
~Coinflip Basic coinflip (50/50 chance).
~Color This cmd sends a message with the color of your name.
~Google U can search for anything and the bot returns the google search request.
~Help This cmd gives you a link to this repository.
~Massping This cmd pings every user in the channel (only works if u are a mod or the broadcaster).
~Math This cmd solves any mathematical calculation.
~Ping Ping cmd with current uptime.
~Pyramid This cmd creates a pyramid of emotes in chat.
~Random chat line (rl) The bot returns a random chat line, which is logged in the DB.
~Rando(random number) The bot gives out a random number. U can also type a range of two numbers. The bot will give u a random number in the given range.
(The range is limited to 2,147,483,647 (max range of Int32)).
~Vanish This cmd deletes your previous messages.
~Your mom Returns a random funny mom joke based on YourMom.json.

Passive Actions
The bot has various passive actions listed down here:

Message interruptions If any user wants to type a four piece emote, the bot breakes the emote combo.
User specific actions If some specific users send a message, the bot returns a funny emote after the message.
ALERT Responds to pajbot in pajlada's chat.
